# Tổng quan dự án
Dự án tập trung phân tích và tối ưu hệ thống quản lý kho cho cửa hàng Bách Hóa Xanh.
Các nghiệp vụ chính:
- Quản lý nhập kho
- Quản lý xuất kho
- Kiểm kê tồn kho
- Phân quyền người dùng
- Báo cáo tồn kho
